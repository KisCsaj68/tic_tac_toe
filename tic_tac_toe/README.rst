tic_tac_toe
===============

tic tac toe game

Usage
-----
To run the tests, coverage report and the static code analyzers, type
```
tox
```

Installation
------------
```
pip install -e .
```

Requirements
^^^^^^^^^^^^
You will need tox installed in your local .venv

Compatibility
-------------
This project requires python 3.9
